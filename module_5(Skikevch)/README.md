Описание проекта.
Соревнование на kaggle: https://www.kaggle.com/c/sf-dst-car-price-prediction

Основные цели и задачи проекта:
Цель: В данном соревновании требуется обучить модель предсказывать стоимость автомобиля по его характеристикам.
Задачи: 
- обработать данные
- протестировать разные алгоритмы
- предсказать таргет с помощью лучшевсего показавшей себя модели.
